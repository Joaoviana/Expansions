## Study Snippet

```js
// (String, 'a string', 'see')
'a string'.charAt(2) + 'ee';

'a string'.charAt(2)
//(String, 's')
's' + 'ee'
//(String, 'see')
```

charAt(x) Method takes a string and returns the char at index 'x'. 
The '+' operator appends both strings together.
